Kérlek másold ide a ikonképet a következő névvel:

app_icon.png

Ajánlott méret: 1024x1024 px (PNG). Átlátszó háttér megengedett.

Miután feltöltötted a fájlt, futtasd a következő parancsokat a projekt gyökérkönyvtárában:

flutter pub get
flutter pub run flutter_launcher_icons:main

Ez generálja az Android és iOS launcher ikonokat automatikusan.
